# Supported Network Protocols

## Available Protocols

The network module supports the following protocols:

1. **TCP** - Transmission Control Protocol for reliable connections
2. **UDP** - User Datagram Protocol for fast, connectionless communication
3. **HTTP** - Hypertext Transfer Protocol for web traffic
4. **HTTPS** - HTTP with SSL/TLS encryption
5. **WebSocket** - Bidirectional communication protocol

## Protocol Selection

Configure the protocol in your network settings:

```yaml
protocol: HTTPS
port: 443
```
